### UnityFramework



#### ------------------------------------------ Under Development --------------------------------------------



##### UnityFramework is built on top of <a href="https://github.com/Alex-Rachel/TEngine"><strong>TEngine</strong></a>, an excellent lightweight Unity framework that integrates both <a href="https://github.com/focus-creative-games/hybridclr"><strong>HybridCLR</strong></a> and <a href="https://github.com/tuyoogame/YooAsset"><strong>YooAsset</strong></a>. Inspired by the design of [WPF](https://github.com/dotnet/wpf), it restructures the architecture using MVVM and an Entity-Component pattern. The core process has been moderately streamlined.

##### Full MVVM data binding support is implemented, with an updated TimeModule and an event module that supports MVVM-style message dispatching.

##### The `main` branch includes a red point system and onboarding guide extracted from Leihuo tools, but they have not yet been deeply refactored.

##### A lightweight version of the EGamePlay skill system is integrated (currently adapted with minimal changes), with plans for deeper customization in the future.

##### A code generator for View and ViewModel modules is implemented. Right-click on a prefab in the Hierarchy panel and select: `Script Generator → GenerateView / GenerateViewModel` (follows custom component naming conventions).

##### See the `demo` folder for usage examples.

##### Basic interfaces and implementation of a GOAP (Goal-Oriented Action Planning) system have been provided and will be expanded further.

##### If you encounter any issues during use, feel free to submit an issue — we will prioritize fixing them.

### Recommended Projects

##### <a href="https://github.com/Alex-Rachel/TEngine"><strong>TEngine</strong></a> - A simple (beginner-friendly, ready-to-use) yet powerful full-platform Unity framework. Ideal for developers or teams looking for a fast-start, well-documented, high-performance, and extensible commercial solution.

##### <a href="https://github.com/dotnet/wpf"><strong>WPF</strong></a> - A UI framework for building Windows desktop applications. WPF supports a wide range of features including app models, resources, controls, graphics, layout, data binding, and document functionality using XAML for declarative UI programming.

##### <a href="https://github.com/focus-creative-games/hybridclr"><strong>HybridCLR</strong></a> - A feature-complete, zero-cost, high-performance, low-memory hotfix solution for Unity that brings native C# runtime support across all platforms.

##### <a href="https://github.com/tuyoogame/YooAsset"><strong>YooAsset</strong></a> - A commercial-grade asset management system validated in games with millions of DAU.

##### <a href="https://github.com/m969/EGamePlay"><strong>EGamePlay</strong></a> - A battle system designed based on the Entity-Component pattern.

##### <a href="https://github.com/crashkonijn/GOAP"><strong>GOAP</strong></a> - A robust, multithreaded Goal-Oriented Action Planning system for Unity, enabling developers to implement complex AI behavior with ease.

##### <a href="https://github.com/vovgou/loxodon-framework"><strong>Loxodon Framework</strong></a> - A lightweight MVVM framework tailored for Unity3D development, inspired by the MVVM patterns of WPF and Android.
